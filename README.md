# یادگیری Vue با سوالات

اپلیکیشن فارسی (RTL) برای یادگیری JavaScript، TypeScript، Vue و اکوسیستم آن.

## اجرا (Bun)

```bash
bun install
bun run dev
bun run build
bun run test:unit
```

## ویژگی‌ها

- ۹ دوره آموزشی با سوالات JSON
- آزمون عادی + آزمون رفع اشتباهات (یکجا)
- پیشرفت در `localStorage`
- UI بر پایه Ant Design Vue

## مسیرها

| مسیر | توضیح |
|------|--------|
| `/` | داشبورد |
| `/learn/:moduleId` | فصل‌های دوره |
| `/learn/:moduleId/:sectionId/quiz` | آزمون |
| `/learn/:moduleId/:sectionId/wrong` | آزمون رفع اشتباهات |
